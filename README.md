# Data-Brain
An AI-powered data science assistant that lets you upload a dataset, explore it interactively, and chat with an intelligent agent that understands your data. Built with Streamlit, LangChain, OpenAI, and SentenceTransformers.
Data Brain — Al-Assisted Data Science & Exploratory Data Analysis
Data Brain is an Al-powered tool designed to streamline data science, EDA, and ML workflows through an intuitive conversational interface.
Users can upload a dataset, visualize it, analyze it, and ask natural-language questions — all inside a single integrated application.
This project combines:
• Streamlit (frontend UI)
• Pandas + Matplotlib (EDA)
• SentenceTransformers (vector embeddings)
• LangChain + OpenAl (Al chat assistant)
